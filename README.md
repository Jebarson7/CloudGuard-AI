# CloudGuard AI

CloudGuard AI is an AI-powered cloud operations monitoring dashboard built using Streamlit and Plotly.

## Features

- CPU Monitoring
- Memory Monitoring
- Network Monitoring
- System Health Score
- AI Risk Analysis
- Prediction Engine
- Alert History
- Incident Report Generation
- Report Download

## Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
